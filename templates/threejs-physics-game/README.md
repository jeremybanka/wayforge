# Three.js Physics Game

A Vite + Solid + Three.js starter with `atom.io` tracking gameplay stamina for jumping.

## What it shows

- a canvas-first 3D scene with directional light, fog, and a simple arena
- a controllable cylinder character with WASD movement and space-to-jump
- manual gravity with a grounded check and jump impulse
- stamina stored in `atom.io`, spent on jump, and regenerated over time

## Local development

```bash
bun install
bun run dev
```
