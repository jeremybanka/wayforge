# atom.io docs

This package owns the docs it ships, but it only ships the generated agent
corpus that is useful inside an installed project.

- `agent/` is generated during `bun run build:agent-docs`.
- Start with `agent/README.md` when using an AI agent in an installed `atom.io` project.
- The authored source lives in the atom.io repository at `packages/atom.io/docs/source`.
